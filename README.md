# تطبيق شُور علينا 🌟
## 📝 وصف التطبيق
تطبيق شُور علينا منصة رقمية مبتكرة تربط بين المتقاعدين أصحاب الخبرات الواسعة والشركات الناشئة التي تبحث عن استشارات عملية موثوقة.
يقوم التطبيق على الاستفادة من الخبرات التراكمية للمتقاعدين وتحويلها إلى قيمة مضافة تساعد الشركات على النمو والتطور.

 ## 🎯 أهداف التطبيق

- تمكين المتقاعدين من الاستمرار في العطاء وتقديم استشارات مبنية على خبراتهم الطويلة
- دعم الشركات الناشئة بتوفير مرشدين ومستشارين ذوي خبرة عملية بأسعار رمزية
- تعزيز الاقتصاد الوطني من خلال الاستفادة من رأس المال البشري وتحويله إلى مورد معرفي منتج


## 💡 القيمة المضافة
### للمتقاعدين 👨‍💼

- تعزيز الدور المجتمعي والاستمرار في الإنتاجية
- توفير دخل إضافي من الخبرات المكتسبة

### للشركات 🏢

- الحصول على استشارات عالية الجودة بأسعار مناسبة
- تجنب الأخطاء المكلفة من خلال الاستفادة من التجارب السابقة

## 🛠️ التقنيات

- Flutter (Dart)
- Firebase Authentication
- Cloud Firestore
- SharedPreferences

---

## 🚀 تشغيل المشروع
1. تأكد من تثبيت Flutter 3.x و Dart.
2. اربط مشروعك مع Firebase وضع ملفات الإعداد (لا ترفعها للمستودع):
   - `android/app/google-services.json`
   - `ios/Runner/GoogleService-Info.plist`




     
  ## 🚀 تشغيل التطبيق
  
```bash
flutter pub get
flutter run
```
