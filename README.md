# تطبيق شُور علينا

تطبيق شُور علينا يهدف إلى إنشاء منصة رقمية مبتكرة تعمل على الربط بين المتقاعدين أصحاب الخبرات الواسعة في مختلف المجالات، وبين الشركات الناشئة والصغيرة التي تبحث عن استشارات عملية موثوقة تساعدها على النمو والتطور. 

تقوم فكرة التطبيق على مبدأ الاستفادة من الخبرات التراكمية للمتقاعدين وتحويلها إلى قيمة مضافة يمكن للشركات استثمارها في تطوير أعمالها، مما يسهم في تحقيق الاستدامة والابتكار في بيئة الأعمال. 

---

## 🛠️ التقنيات
- Flutter (Dart)
- Firebase Authentication
- Cloud Firestore
- SharedPreferences

---

## 🚀 تشغيل المشروع
1. تأكد من تثبيت Flutter 3.x و Dart.
2. اربط مشروعك مع Firebase وضع ملفات الإعداد (لا ترفعها للمستودع):
   - `android/app/google-services.json`
   - `ios/Runner/GoogleService-Info.plist`##
   - 🚀 تشغيل التطبيق
```bash
flutter pub get
flutter run
```
